## npm Installation

### Prerequisites

- Node.js version 18.10 or above
- npm version 8 or above

### Install n8n globally

```bash
npm install n8n -g
```

### Start n8n

```bash
n8n
```

This will start n8n and open it in your default browser at `http://localhost:5678`.

## Docker Installation

### Quick Start

```bash
docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  n8nio/n8n
```

### With Data Persistence

```bash
docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  n8nio/n8n
```

### Using Docker Compose

```yaml
version: '3.8'

services:
  n8n:
    image: n8nio/n8n
    ports:
      - "5678:5678"
    environment:
      - N8N_BASIC_AUTH_ACTIVE=true
      - N8N_BASIC_AUTH_USER=admin
      - N8N_BASIC_AUTH_PASSWORD=password
    volumes:
      - ~/.n8n:/home/node/.n8n
```

## Environment Variables

Customize your n8n installation with environment variables:

```bash
# Basic authentication
export N8N_BASIC_AUTH_ACTIVE=true
export N8N_BASIC_AUTH_USER=admin
export N8N_BASIC_AUTH_PASSWORD=password

# Change default port
export N8N_PORT=8080

# Set custom data directory
export N8N_USER_FOLDER=/path/to/data

# Enable webhook URL
export WEBHOOK_URL=https://your-domain.com/
```
